Enumerable#map

Learn about the enumerable map method

November 1, 2015

Ruby's enumerable module has many built-in methods that make life easier for a coder. They also make code more readable and therefore easier to understand.

One particularly helpful enumerator is map. Map iterates over every element in an enumerator (which can be a range of numbers or an array) and returns the results in an array. So, it basically goes through a chunk of information you provide and does the same thing to each piece of information and returns it in an array.

Above, you can see the mapping of a range of integers. Calling map on the range adds 10 to each integer, and returns an array of those integers.

Another example above is mapping an array of strings. This looks more similar to a regular "each" iteration, but it returns a new array out of objects other than arrays.

Map is a great iterator if you want to take information and turn it into a new array. For more info, the Ruby documentation website always has the best info.
