CSS: Display inline vs. inline block

Learn the difference between CSS's display inline and inline block

October 18, 2015

CSS can be confusing. Two useful elements for positioning text or divs are display inline or inline block.

Display: inline means that elements are in a line but further formatting doesn't work. For example, each of these divs are set to display: inline.

When those same divs are set to display: inline-block, here is the result. Div 2 had a "width" set and Div 4 had a "height" set. Those two properties are revealed when we change their displays to inline-block.

Inline-block also allows top and bottom margins and padding, which inline does not.

So, inline-block elements add the two following things to inline elemments: they allow top and bottom padding and margins and set widths and heights.

If you're ever in doubt, you can use Chrome DevTools (right-click your window and choose "inspect element") to test a page's elements with different CSS styling.
